Visibility

Hiding elements can be achieved in a few ways with different purposes.

  • display: none - this fully remove the element from the DOM and is a good default, but be careful using this with animations/transitions

  • visibility: hidden - this keeps the element in the DOM and document flow (i.e., there is still space held for it in the layout) but simply hides rendering it (useful for animations/transitions)

    NOTE: Also hides content from screen readers.

  • hidden HTML attribute - NOT recommended since display values in CSS will override it.
  • content-visibility: hidden - this is a very new CSS property and is not recommended at the moment, due to limited support.

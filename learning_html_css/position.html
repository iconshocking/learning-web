Basic document flow

I am a basic block level element. My adjacent block level elements sit on new lines below me.

By default we span 100% of the width of our parent element, and our height is as tall as our child content. Our total width and height is our content + padding + border width/height.

We are separated by our margins. Because of margin collapsing, we are separated by the width of one of our margins, not both.

inline elements like this one and this one sit on the same line as one another, and adjacent text nodes, if there is space on the same line. line-height can also used to center text vertically by setting the height to the line-height, but this should only be used as necessary because line-height when a percentage is based off the font-size of the element, not the height of the element. So this should only be used with elements of hardcoded height.

Priority loading

dns-prefetch

Performs DNS resolution, so only useful for different origins.

preconnect

Performs DNS resolution, TCP connection, and TLS handshake. Only use this and preload for critical resources to avoid bogging down important downloads with establishing less important connections.

preload

Prioritizes downloading of a resource, but does not load it until referenced by other code. Ideal for preloading fonts or other assets that may not be discovered until later in the HTML.

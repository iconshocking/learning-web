• Unrecognized type values for input elements degrade to the text type
  • Generally use as little CSS styling as possible for form fields since they are so difficult and inconsistent to style.
  • Follow unobtrusive JS principles where the site should still function as much as possible if a script breaks.
  • Polyfills can be slow on legacy browsers, so be wary of using too many or ones that are too heavy.
